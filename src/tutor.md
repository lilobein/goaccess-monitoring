## Необходимые действия для веб-версии
1) sudo apt install goaccess
2) запустить скрипт main.sh
3) перейти по http://localhost:9304/goaccess_report_by_log.html

## Если используются различные варианты, то после исполнения скрипта с параметром (1-4) необходимо перейти по соответствующим ссылкам:
- http://localhost:9304/goaccess_report_by_log_var1.html
- http://localhost:9304/goaccess_report_by_log_var2.html
- http://localhost:9304/goaccess_report_by_log_var3.html
- http://localhost:9304/goaccess_report_by_log_var4.html
